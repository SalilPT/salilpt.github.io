Prickly Testers is a card game I made in 2020. It's about strategically attacking and defending against other players using cards drawn from a shuffled deck until only one player remains.


The itch.io page for it is here.

How It Was Made

I made a color palette using color.adobe.com. The colors I used for that palette were #76C5FA, #50C1D9, #51C2BA, #348C70, and #338A59. I chose these because they reminded me of cacti and laboratories, which were related to my idea for the game at the time. All of the art for the game was made using Adobe Illustrator, which means that it uses only vector graphics.

To play it, I used playingcards.io to create and customize a room suited to play the game digitally. Since that site allows for online play, I was able to playtest it with friends and classmates.

List of Scrapped Rules and Mechanics

  • In the original prototype, the counterattack value of a card was dealt in full back to an attacker.
  • At one point in development, there was a "Nullify" option which would allow a defender to cancel all incoming damage.
  • Attacking players did not have the option to discard a card and then end their turn. Attacks were always required.
  • The starting HP of players used to be the number of players times 8. This made games go on for too long.
  • Attack/Defense values didn't go all the way to 9. Also, there were no cards with 0 Counterattack.
